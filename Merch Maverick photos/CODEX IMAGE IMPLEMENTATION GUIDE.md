## FILE FORMAT RECOMMENDATION: PNG vs WebM

### **PNG Files (Current)**
✅ **Best for:** Static images, product photos, headshots, factory images  
✅ **Advantages:**
- Lossless compression (no quality loss)
- Supports transparency
- Excellent for detailed photography
- Universal browser support
- Perfect for 2K quality images

❌ **Disadvantages:**
- Larger file size than WebP
- Not ideal for video content

### **WebM Files (Alternative)**
❌ **Not recommended for static images**  
✅ **Best for:** Video content only (3D videos, animations)

### **MY RECOMMENDATION**
**Keep PNG files for all static images.** They're perfect for your 2K quality photos. WebM is only for video content (which you'll add later for the 3D animation).

**For web optimization, consider converting PNG to WebP** (modern format, 25-35% smaller file size, same quality). Codex can do this during implementation.

## TECHNICAL SPECIFICATIONS FOR CODEX

### **Image Sizing Guidelines**

**Hero Images (16:9 aspect ratio):**
- Desktop: 1920px × 1080px
- Tablet: 768px × 432px
- Mobile: 375px × 211px

**Section Images (4:3 aspect ratio):**
- Desktop: 800px × 600px
- Tablet: 600px × 450px
- Mobile: 375px × 281px

**Card Images (1:1 aspect ratio):**
- Desktop: 300px × 300px
- Tablet: 250px × 250px
- Mobile: 100% width (responsive)

**Testimonial Avatars (1:1 aspect ratio):**
- Desktop: 100px × 100px
- Tablet: 80px × 80px
- Mobile: 60px × 60px

### **CSS Responsive Classes (Example)**

```css
/* Hero Images */
.hero-image {
  width: 100%;
  height: auto;
  object-fit: cover;
  aspect-ratio: 16 / 9;
}

/* Section Images */
.section-image {
  width: 100%;
  height: auto;
  object-fit: cover;
  aspect-ratio: 4 / 3;
}

/* Card Images */
.card-image {
  width: 100%;
  height: auto;
  object-fit: cover;
  aspect-ratio: 1 / 1;
}

/* Avatar Images */
.avatar-image {
  width: 100px;
  height: 100px;
  border-radius: 50%;
  object-fit: cover;
}

@media (max-width: 768px) {
  .hero-image { height: 432px; }
  .section-image { height: 281px; }
  .avatar-image { width: 80px; height: 80px; }
}

@media (max-width: 375px) {
  .hero-image { height: 211px; }
  .section-image { height: 281px; }
  .avatar-image { width: 60px; height: 60px; }
}
```
